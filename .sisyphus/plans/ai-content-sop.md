# AI 內容生成 SOP（5步驗證法）

> **用途**: 指導所有 AI 內容生成和驗證流程
> **適用範圍**: myo-hk.github.io 所有文章
> **最後更新**: 2026-04-02

---

## 5 步驗證法流程

### Step 1: 關鍵字研究（AI 輔助）

**工具**: Exa Search
**時間**: 15-30 分鐘

1. 用 Exa 搜尋長尾關鍵字
   ```
   websearch_web_search_exa(
     query="香港結婚 [主題] 關鍵字 搜尋量",
     numResults=10
   )
   ```
2. 確定 1 個主關鍵字 + 3-5 個副關鍵字
3. 記錄搜尋意圖（資訊型/商業型/交易型）

### Step 2: 官方資料收集（人工）

**工具**: Web Fetch + 手動瀏覽
**時間**: 30-60 分鐘

1. 從官方來源收集資料：
   - `immd.gov.hk` — 婚姻登記程序
   - `elegislation.gov.hk` — 法例條文
   - `clic.org.hk` — 法律解釋
2. 標註來源 URL 和最後更新日期
3. 記錄具體數字（費用、日期、表格編號）

### Step 3: AI 生成文章框架

**工具**: AI（ChatGPT/Claude/OpenWork）
**時間**: 5-10 分鐘

**Prompt 模板**:
```
你是 SEO 專家和香港婚慶顧問。請為關鍵字「[主關鍵字]」生成文章大綱。

要求：
1. H1 標題（包含主關鍵字）
2. H2/H3 結構（8-12 個小節）
3. FAQ 區塊（5 個問題）
4. 標註哪些段落需要填入官方資料
5. 不要生成具體法律內容，只生成框架

文章主題：[主題描述]
目標讀者：[讀者描述]
搜尋意圖：[資訊型/商業型/交易型]
```

### Step 4: 人工填寫 + 驗證

**工具**: 手動編輯
**時間**: 60-120 分鐘

1. 填入 Step 2 收集的官方資料
2. 每個法律/程序聲明都要有來源註釋
3. 加入個人經驗/產品推薦
4. 加入內部連結（Pillar → Cluster, Cluster → Cluster）
5. 加入免責聲明（HTML 模板見下方）
6. 加入 CTA（WhatsApp/Instagram）

### Step 5: 上線前審核

**工具**: 檢查清單
**時間**: 15-30 分鐘

使用下方「內容驗證清單」逐項檢查。

## AI Prompt 模板（按文章類型）

### 官方程序教學類

```
你是 SEO 專家和香港婚慶顧問。請為關鍵字「[關鍵字]」生成文章框架。

要求：
- H1/H2/H3 結構
- 包含步驟清單
- 包含 FAQ（5 個問題）
- 標註需要官方資料的位置
- 不要生成具體法律內容

文章主題：[主題]
```

### 產品推薦/比較類

```
你是 SEO 專家和產品顧問。請為關鍵字「[關鍵字]」生成文章框架。

要求：
- H1/H2/H3 結構
- 包含 5 個產品比較
- 包含比較表格
- 包含購買建議
- 加入產品自然植入點

文章主題：[主題]
```

### FAQ 整理類

```
你是 SEO 專家和客服顧問。請為關鍵字「[關鍵字]」生成 FAQ 文章框架。

要求：
- H1 標題
- 10 個常見問題（H2）
- 每個答案 100-200 字
- 標註需要官方核對的答案
- 加入相關文章連結

文章主題：[主題]
```

## 內容驗證清單

### 法律/程序聲明
- [ ] 有 Tier 0 或 Tier 1 來源支持
- [ ] 註明來源 URL 和查證日期
- [ ] 加入免責聲明

### 價格/費用聲明
- [ ] 查證最新官方價格
- [ ] 註明「價格可能變動」

### 產品/服務聲明
- [ ] 多個來源交叉驗證
- [ ] 避免絕對性描述

### SEO 檢查
- [ ] 標題包含主關鍵字
- [ ] meta description 150-160 字元
- [ ] 內部連結至少 3 個
- [ ] 圖片有 alt 屬性
