## Tier 0 — 原始法例（最高權威，不可被推翻）

| 來源 | URL | 可信度 | 說明 |
|------|-----|--------|------|
| **雙語法例系統** | `elegislation.gov.hk` | ⭐⭐⭐⭐⭐ | 香港政府官方雙語法例資料庫，所有法例的權威來源 |
| **立法會憲報** | `legco.gov.hk` | ⭐⭐⭐⭐⭐ | 法例原始頒布文件 |
| **政府憲報** | `gazette.gov.hk` | ⭐⭐⭐⭐⭐ | 法定公告、修訂通知 |

**驗證要求**: 所有法律條文聲明必須引用 Tier 0 來源。

## 三角交叉驗證法

**規則**: 任何「事實聲明」必須至少 2 個不同 Tier 的來源交叉驗證。

### 驗證流程

```
1. 識別聲明類型（法律/程序/價格/產品/經驗）
2. 找到最高 Tier 的來源支持
3. 用另一個 Tier 的來源交叉驗證
4. 如果矛盾 → 以高 Tier 為準，註明差異
5. 如果一致 → 可以發布，註明來源
```

## 實際案例分析：「香港結婚證書是 A4 尺寸」

### 搜尋結果

| 來源 | Tier | 說法 | 可信度 |
|------|------|------|--------|
| elegislation.gov.hk | Tier 0 | **未明確規定尺寸** | 最高 |
| immd.gov.hk | Tier 1 | 未明確提及 | 高 |
| 婚姻監禮人 PDF | Tier 2 | MediaBox ≈ A4 (209×298mm) | 中 |
| 我的 Threads | Tier 4 | A4（210×297mm） | 低 |

### 結論

法例沒有規定證書尺寸。所謂「A4」是實務慣例，不是法律要求。

### ✅ 正確寫法

> 「香港婚姻登記處發出的結婚證書尺寸約為 A4 大小（實測約 209mm × 298mm），因此選購證書套時建議選擇能容納 A4 尺寸的款式。」
> 來源：婚姻監禮人文件樣本 + 實測

### ❌ 錯誤寫法

> 「政府規定結婚證書是 A4 尺寸」
> （法例沒有這樣規定）

## 內容驗證清單（每篇文章必做）

### 法律/程序聲明
- [ ] 是否有 Tier 0 來源（elegislation.gov.hk）支持？
- [ ] 是否有 Tier 1 來源（immd.gov.hk / clic.org.hk）交叉驗證？
- [ ] 是否註明了來源 URL 和最後查證日期？
- [ ] 是否加入了免責聲明？

### 價格/費用聲明
- [ ] 是否查證了最新官方價格？
- [ ] 是否註明了「價格可能變動，以官方最新公佈為準」？

### 產品/服務聲明
- [ ] 是否有多個商業來源交叉驗證？
- [ ] 是否避免了絕對性描述（如「最好」「最平」）？
- [ ] 是否加入了「價格以商家最新公佈為準」？

### 個人經驗聲明
- [ ] 是否明確標示為「個人經驗」而非「事實」？
- [ ] 是否避免了將個人經驗泛化為普遍規律？

## 多源驗證工具指南

### Exa Search
- **用途**: 語義搜尋，找最新資訊和競爭情報
- **用法**: 用自然語言描述理想結果頁面
- **例子**: `websearch_web_search_exa(query="香港結婚證書套 推薦 價錢 2026", numResults=10)`

### Web Fetch
- **用途**: 直接抓取指定 URL 內容
- **用法**: 提供完整 URL，返回 markdown 或 text
- **例子**: `webfetch(url="https://www.immd.gov.hk/hkt/marriage/Registration_of_a_Marriage.html", format="markdown")`

### 交叉驗證流程
1. Exa 搜尋找多個來源
2. Web Fetch 抓官方內容
3. 比對矛盾點
4. 以高 Tier 為準
