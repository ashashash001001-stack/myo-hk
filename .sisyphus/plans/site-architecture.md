## Pillar Page（核心頁）

### 主題
「2026 香港結婚完整攻略：由註冊到婚宴的每一步」

### SEO 關鍵字
- **主關鍵字**: 香港結婚攻略
- **副關鍵字**: 香港結婚流程, 香港結婚手續, 香港註冊結婚, 結婚準備清單

### 內容大綱

```
H1: 2026 香港結婚完整攻略：由註冊到婚宴的每一步

H2: 一、結婚前準備
  H3: 確認結婚資格（年齡、國籍、婚姻狀況）
  H3: 準備所需文件清單
  H3: 預算規劃（註冊費、婚宴、婚戒等）

H2: 二、遞交擬結婚通知書
  H3: 何時遞交（15 天至 3 個月前）
  H3: 何處遞交（婚姻登記處 vs 婚姻監禮人）
  H3: 費用（HK$305）
  H3: 所需文件

H2: 三、選擇婚禮場地
  H3: 婚姻登記處（大會堂、紅棉路等）
  H3: 特許禮拜場所
  H3: 婚姻監禮人主持

H2: 四、婚禮當日流程
  H3: 註冊程序
  H3: 見證人要求
  H3: 結婚證書簽發

H2: 五、婚後事項
  H3: 更改身份證婚姻狀況
  H3: 申請結婚證書核證副本
  H3: 稅務優惠（已婚人士免稅額）

H2: 六、常見問題 FAQ
  H3: 海外人士可以在香港結婚嗎？
  H3: 離婚人士再婚需要什麼文件？
  H3: 結婚證書尺寸是多少？

H2: 七、延伸閱讀
  → 連結到所有 Cluster Pages
```

### 產品植入點
- 在「婚禮當日流程」中自然植入：「註冊後，用我們的專屬證書套珍藏您的結婚證書」
- CTA: 「立即定制專屬證書套 → WhatsApp 聯絡我們」

## SEO URL 命名規範

| 規則 | 例子 |
|------|------|
| 使用中文檔名 | `香港結婚完整攻略.html` |
| 避免特殊字符 | 不用 `?`, `&`, `#` |
| 小寫優先 | 但中文不受影響 |
| 包含關鍵字 | `結婚證書套推薦.html` |
| 保持簡短 | 不超過 20 個中文字 |
